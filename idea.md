## Main Idea
If any person has lots of documents stored in some folder and he/she wants to search for the relevant document but can't remember the exact name, they can type in the relevant words that they remember from the document and based on that, giving suggested matching documents.

For this ML algorithms will be used. 

Tech Stack:
Backend: OOPs with typescript
Frontend: typescript, shadcn
ML algorithms: Langgraph, RAG system.


## Scope
Where lots of documents are there in a folder.

## Key Feature
Put in context, get relevant document.
